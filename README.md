# Modern Sudoku

A clean, responsive, and feature-rich Sudoku game built with modern web technologies.

## Features

- **Modern UI**: Clean, minimalist design that focuses on gameplay.
- **Responsive**: Fully playable on mobile, tablet, and desktop devices.
- **Dark Mode**: Toggle between light and dark themes.
- **Game Modes**: Easy, Medium, and Hard difficulty levels.
- **Tools**:
  - **Undo**: Quickly revert your last move.
  - **Erase**: Clear a specific cell.
  - **Notes**: (Visual placeholder for future implementation).
- **Timer**: Track how long it takes you to solve the puzzle.
- **Smart Highlights**:
  - Highlights the current row, column, and box.
  - Highlights all instances of the selected number.
  - Real-time error detection for obvious conflicts.

## How to Play

1. Open `index.html` in your web browser.
2. Select a difficulty level.
3. Click on a cell to select it.
4. Enter a number (1-9) using your keyboard or the on-screen keypad.
5. Use the tools (Undo, Erase) as needed.
6. Complete the grid so that every row, column, and 3x3 box contains the digits 1 through 9.

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, CSS Variables)
- JavaScript (ES6+)
